This folder contains examples of useful technical notes and learning materials in the subjects of non traditional machining processes, materials modelling, metal cutting experimental process and simulation, and surface integrity.
It is fully written by Dr. Eng. Lamice Denguir during the execution of her projects in the purpose of documenting engineering processes, instructing collaborators, or teaching students.
The folder is organized as follows:

  - Courses: it contains the support of the course "Non Traditional Machining Process divided into 3 parts. The first and second parts are available as video courses on YouTube following this link: https://youtu.be/4UiY-csUxj8 (for part 1) and this link: https://youtu.be/kXlyXt-4X6k (for part 2).
  
  - Tutorials: the available tutorials are devided into 3 groups: 
  
    - material modelling: 2 tutorials are available: Elasto Plastic Non Linear material model parameters identification procedure (EPNL_parameters_identification_procedure.pdf) and Elasto-visco-Plastic material model parameters identification procedure (EVP_parameters_identification_procedure.pdf).
    
    - metal cutting: this folder is divided into 2 sub-folders:
      - Experimental process: it contains tutorials about: cutting OFHC copper in planing configuration, measuring the chip compression ratio, milling forces calculus using DasyLab, and tool flank wear trials protocol.
      - Simulation: it contains one tutorial about ALE modeling applied to orthogonal cutting of OFHC copper, and one tutorial about metal cutting simulation using Abaqus.
      
    - surface integrity: This folder contains a tutorial about metal specimen preparation for micrography, and a full extended document about multi-physical analysis of corrosion resistance.

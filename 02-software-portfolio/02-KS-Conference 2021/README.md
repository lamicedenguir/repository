# 🎓 KS-Conference

> **A secure conference streaming and replay platform for scientific events.**

KS-Conference is a web platform designed to help conference organizers manage online access to scientific presentations, live streams, and recorded sessions in a secure and user-friendly environment.

Originally developed as part of the **Knowledge & Skills** initiative, the platform aims to extend the reach of academic conferences while keeping access restricted to registered participants.

🌐 **Live Demo:** https://ks-conference.vercel.app/

---

# Overview

Scientific conferences increasingly rely on online participation and recorded presentations. However, many organizers struggle to provide a centralized, secure, and professional platform where attendees can easily access conference materials.

KS-Conference addresses this challenge by providing a dedicated portal where participants can:

- Watch live conference sessions
- Replay recorded presentations
- Access conference materials
- Authenticate securely
- View only the content they are authorized to access

---

# Features

## 📺 Conference Replay

Allow registered participants to watch recorded conference presentations after the event.

---

## 🎥 Live Streaming

Redirect attendees to live conference sessions using integrated streaming channels.

---

## 🔒 Secure Access

Conference organizers control:

- participant registration
- access permissions
- availability period of conference content

Only authorized users can access protected material.
